Click the box, I dare you. You knew it would do something. Here the box is receiving a click event. What's really important here is how the box gets it click event. In this case the event was added via JS, not inline via the attribute. Looks closely and confirm this for yourself.

Even more importantly is how the box targets itself to add the special class. When the function is invoked, by clicking the box. The box targets itself using the keyword this. Which is a very special word in JS. Since the function added to the box object, when that function is invoked this will always refer to the owner of the function.
